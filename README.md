# TP53 Conservation Analysis

## Overview

This project investigates the evolutionary conservation of the TP53 tumor suppressor protein across multiple vertebrate species using bioinformatics approaches. TP53 plays a critical role in maintaining genomic stability, regulating the cell cycle, and preventing tumor development.

## Objectives

* Retrieve TP53 protein sequences from multiple vertebrate species.
* Perform multiple sequence alignment to identify conserved and variable regions.
* Analyze sequence conservation patterns across species.
* Construct and interpret a phylogenetic tree.
* Explore the evolutionary significance of TP53 conservation.

## Species Included

* Human (*Homo sapiens*)
* Mouse (*Mus musculus*)
* Rat (*Rattus norvegicus*)
* Dog (*Canis lupus familiaris*)
* Chicken (*Gallus gallus*)
* Zebrafish (*Danio rerio*)

## Tools and Resources

* NCBI Database
* Clustal Omega
* MEGA Software
